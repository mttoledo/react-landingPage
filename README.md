# Simple Landing Page

A responsive Landing Page built to showcase advanced React concepts, modern styling with Tailwind CSS, and fluid animations using Framer Motion.

## Tech Stack

This project was built using the most modern tools in the Front-end ecosystem:

- React.js: Core library for building the user interface.
- Tailwind CSS: Utility-first CSS framework for rapid and responsive design.
- Framer Motion: Powerful animation library for smooth transitions and scroll-based interactions.
- Lucide React: Lightweight and customizable icon set.
- Vite: Extremely fast build tool for a modern development workflow.
